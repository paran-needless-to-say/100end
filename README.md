# Trace-X Backend

Blockchain transaction analysis system for tracking fund flows and cross-chain transactions.

## Setup

```bash
pip3 install -e .
```

## Configuration

Set your Etherscan API key:

```bash
export ETHERSCAN_API_KEY=your_api_key_here
```

## Run

```bash
python3 main.py
```

Server runs at `http://localhost:8888`
